Scratch

In this lesson, we’re going to learn how to create different effects! In the purple sections, find the “change effect by” block.

Using this button, we can change the color, create mosaics, pixelate, or create effects like whirl and ghost.

Now let’s try this with your sprite!

Let’s try applying multiple effects and add some sound to the image effects.

Those three blocks are: Change () Effect by () — This changes the strength of a chosen effect Set () Effect to () — This sets the strength of a chosen effect Clear Graphic Effects — This clears all effects over one sprite or the Stage
