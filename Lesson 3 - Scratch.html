Scratch

1. First open a new scratch project or you are welcome to work on an existing one.

2. Select a sprite that you would like to change the color of.

3. Create a condition, by selecting the button labeled Control on the left of the screen.

4. Select the condition you would like your sprite to change color.

5. Then, drag that block out into the working station.

6. Next go to the left of the screen and click on the button labeled "Looks".

7. Then, click and drag the button labeled “change color effect by 25”.

8. The color should be in a small box with a drop down menu. You can change this characteristic according to what you want.

9. The 25 should be in a white bubble. You can change this number depending on the different shades of colors you would like for your sprite to change into.
