3.1 BOOLEAN VALUES AND EXPRESSIONS
Boolean Values are very important in Computer Science. They consist of two values: TRUE and FALSE. When using this in a python, it has to exactly capitalized as shown, and the type for these values are bool. Often when wanting the program to return or when using Boolean values, it is common to use the “==” operator. This compares two values. It will return True if the values equal each other and False if they don’t. Remember “=” is the assignment operator. Don’t get them confused! There are a few other comparison operators for example: “!=”, “>”, “<”, “<=”, and “>=”. Make sure that you don’t confuse “=<” or “=>” comparison symbols, because those don’t exist in Python.>
